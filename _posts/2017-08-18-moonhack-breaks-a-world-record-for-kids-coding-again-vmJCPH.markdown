On Tuesday, 28,575 kids around Australia broke a world record by coding on the same day. It was all for Moonhack , an event for ages 8 to 18 set up by Code Clubs Australia, a network of 1,950 volunteer-led, after-school coding clubs for children. This year was the first time the event officially expanded internationally from its native Australia, due in part to a strange occurrence which happened after the first Moonhack last year.
